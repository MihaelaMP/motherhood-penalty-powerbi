
# Motherhood Penalty in Italia 📊
Analisi del tasso di occupazione femminile e maschile in Italia  
in base al numero di figli, realizzata con Power BI.
## 📌 Obiettivo
Verificare se il tasso di occupazione femminile si abbassa  
al crescere del numero di figli e se lo stesso fenomeno  
si osserva per gli uomini.
## 📈 Risultati principali
- Il tasso di occupazione delle donne **scende** al crescere del numero di figli
- Il tasso di occupazione degli uomini **sale** al crescere del numero di figli
- Il gap medio di occupazione tra uomini e donne è di **28 punti percentuali**
- La pandemia Covid (2020) ha accentuato temporaneamente il divario
- Dal 2021 si osserva una lenta ripresa dell'occupazione femminile
## 🗂️ Fonte dati
- **Eurostat** — dataset `lfst_hheredch`  
- Fascia d'età: 25-49 anni  
- Periodo: 2006-2024  
- Paese: Italia
## 🛠️ Strumenti utilizzati
- Power BI Desktop
- Power Query (pulizia dati)
- DAX (misure calcolate)
